burner open

Open a project directory in your editor, file explorer, or get its path.

burner open [name] [options]

Arguments

ArgumentDescription
[name] Project name to open. Interactive mode if not provided.

Options

OptionDescription
-e, --explorer Open in file explorer
-c, --code Open in editor (uses configured editor)
-i, --interactive Interactive mode: select project from list

Examples

# Interactive: select project and action
burner open
# Print project path (useful for scripting)
burner open my-experiment
# Open in editor
burner open my-experiment -c
# Open in file explorer
burner open my-experiment -e
# Change directory to project (bash/zsh)
cd $(burner open my-experiment)

Interactive Mode

When no project name is provided, interactive mode lets you:

  1. Select a project from a list
  2. Choose an action:
    • Open in editor (uses configured editor)
    • Open in file explorer
    • Copy path to clipboard

Tip: Configure your preferred editor with burner config --editor cursor
