burner list

List all burner projects with their age and template information.

burner list [options]

Aliases

burner ls

Options

OptionDescription
-a, --all Show all details including full paths

Examples

# List all projects
burner list
# Using alias
burner ls
# Show full paths
burner list -a

Output

Projects are displayed in a table with:

  • Name - The project folder name (YYMMDD-name format)
  • Template - The template used to create the project
  • Age - Color-coded age indicator:
    • Green - Less than 7 days
    • Yellow - 7-30 days
    • Red - More than 30 days
  • Path (with -a) - Full filesystem path
