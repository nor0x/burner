burner templates

List all available templates (built-in and custom).

burner templates

Examples

# List all available templates
burner templates

Output

Displays a table with:

  • Template - The template name (used with burner new)
  • Filename - The script filename
  • Type - built-in or custom

Built-in Templates

TemplateDescription
dotnet .NET Console Application
web HTML + JavaScript + CSS Web App
