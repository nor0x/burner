burner stats

Show project statistics with visual charts.

burner stats

Examples

# Show statistics
burner stats

Output

The stats command displays:

Overview Panel

  • Total Projects - Number of burner projects
  • Disk Usage - Total disk space used
  • Avg Age - Average age of projects
  • Due for Cleanup - Projects older than auto-clean threshold

Projects by Template

Bar chart showing how many projects use each template.

Projects by Age

Breakdown chart showing age distribution:

  • Less than 7 days
  • 7-30 days
  • 30-90 days
  • More than 90 days

Projects Created (Last 6 Months)

Bar chart showing project creation over time.

Tip: Use stats to identify old projects and decide when to run burner burn --days N.
